# Frontend for Build Your Fund application

## Summary
Project based on Vite, React, Typescript.

## Technologies Stack
- Typescript
- [React](https://reactjs.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

- Clone project
- Install dependencies "npm install"
- run "npm run dev"
